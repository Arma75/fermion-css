📝 HTML 텍스트 관련 태그 설명 및 예제

의미론적 태그의 중요성

HTML 태그들은 단순히 텍스트를 표시하는 것을 넘어, 콘텐츠에 의미를 부여하여 웹 접근성과 검색 엔진 최적화를 돕습니다. 다음은 주요 텍스트 태그들의 설명과 활용 예제입니다.

텍스트 태그들

`h1` ~ `h6` 태그 (Heading)

문서의 제목과 부제목을 나타내는 태그입니다. `h1`이 가장 중요한 제목이며, 숫자가 커질수록 중요도가 낮아집니다.

`p` 태그 (Paragraph)

가장 기본적인 태그로, 텍스트의 단락을 나타냅니다.

이것은 첫 번째 단락입니다. 단락은 브라우저에서 위아래로 여백을 만들어 다른 콘텐츠와 구분됩니다.

두 번째 단락입니다.

`strong` 태그 (Strong Importance)

텍스트의 **중요성**이나 심각성을 강조합니다.

경고: 이 작업은 되돌릴 수 없습니다. 계속하시겠습니까?

`b` 태그 (Bold)

특별한 의미 없이 텍스트를 단순히 **굵게** 표시합니다.

오늘의 최고 인기 상품은 바로 이것입니다. 더 자세한 정보는 링크를 참고하세요.

`em` 태그 (Emphasis)

텍스트를 **강조**하여 발음상의 강세를 나타냅니다.

저는 이 책이 정말 재미있다고 생각해요. 여러분에게 강력 추천합니다.

`i` 태그 (Italic)

특별한 의미 없이 텍스트를 단순히 기울여 표시합니다. 외래어, 기술 용어 등에 사용됩니다.

새로운 소셜 미디어 트렌드는 인스타그래머블입니다.

`abbr` 태그 (Abbreviation)

**약어**를 나타냅니다. `title` 속성을 사용하여 전체 이름을 제공합니다.

웹 접근성은 WA라고도 부릅니다. 또한, 웹사이트의 SEO는 SEO라고 합니다.

`dfn` 태그 (Definition)

**정의하는 용어**를 나타냅니다.

HTML은 웹페이지를 만드는 언어입니다. CSS는 웹페이지를 꾸미는 언어입니다.

`del` 태그 (Deleted Text)

문서에서 **삭제된 텍스트**를 나타냅니다. 보통 취소선이 표시됩니다.

이전 가격: 50,000원 40,000원

`ins` 태그 (Inserted Text)

문서에 **새로 삽입된 텍스트**를 나타냅니다. 보통 밑줄이 표시됩니다.

최신 가격: 40,000원입니다.

`sup` 태그 (Superscript)

**위 첨자**를 나타냅니다. 수학이나 과학 표기법에 사용됩니다.

아인슈타인의 공식: E=mc2

`sub` 태그 (Subscript)

**아래 첨자**를 나타냅니다. 화학식에 자주 사용됩니다.

물의 화학식: H2O

`small` 태그 (Small Print)

저작권, 법적 고지 등 **부수적이고 작은 글**을 나타냅니다.

© 2024. 모든 저작권은 보호됩니다.

`mark` 태그 (Marked Text)

텍스트의 **관련성**을 나타내기 위해 하이라이트합니다.

검색 결과에서 키워드를 찾았습니다.

코드 및 인용 태그

`hr` 태그 (Horizontal Rule)

콘텐츠의 **주제 변화**를 나타내는 수평선입니다.

첫 번째 주제의 끝.

새로운 주제의 시작.

`code` 태그 (Code Fragment)

**컴퓨터 코드 조각**을 나타냅니다.

자바스크립트 변수 선언: let name = '홍길동';

HTML 파일의 기본 구조는 다음과 같습니다
<!DOCTYPE html> <html> <head> <body>

`pre` 태그 (Preformatted Text)

미리 서식이 지정된 텍스트로, **공백과 줄바꿈이 그대로 유지**됩니다. 

def greet(name):
    print(f"Hello, {name}!")

greet("홍길동")

`blockquote` 태그 (Block Quotation)

다른 출처에서 가져온 **긴 인용문**을 나타냅니다.

가장 훌륭한 사람들은 가장 큰 장애물을 만날 때까지 겸손을 유지합니다.

- 랄프 왈도 에머슨

`q` 태그 (Quote)

**짧은 인라인 인용문**을 나타냅니다.

세상의 모든 사람들은 내가 하는 일을 사랑하라는 조언을 자주 듣습니다.

`cite` 태그 (Citation)

인용된 **작품의 제목**을 나타냅니다.

이것은 생각의 탄생이라는 책에서 인용되었습니다.
